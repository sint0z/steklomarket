# Сайт бренда «Стекломаркет»

## Описание проекта
Веб-сайт разработан по договоренности с ООО «Витрум» для бренда «Стекломаркет» (https://steklomarket.ru).  
Проект выполнен на чистом HTML, CSS и JavaScript без использования фреймворков.  
Основная цель — представить продукцию и услуги компании, а также обеспечить удобную коммуникацию с клиентами.

## Технологии
- HTML5 — структура страниц  
- CSS3 — стилизация и адаптивная верстка  
- JavaScript (Vanilla JS) — интерактивность и логика интерфейса  
- Flexbox и Grid Layout — организация контента  
- Минимум внешних зависимостей для повышения производительности

## Особенности
- Полностью адаптивный дизайн  
- Поддержка современных браузеров  
- Удобная навигация  
- Контактные формы с базовой проверкой данных  
- Чистая и понятная структура проекта
